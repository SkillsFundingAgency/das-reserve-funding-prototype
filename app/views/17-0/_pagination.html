Pagination

A component to navigate through multiple pages of a list of items.

The pagination component consists of:

  • links to specific pages
  • links to previous and next page (displayed if applicable)
  • a label for the currently displayed page
  • and a label indicating how many items are displayed on the current page out of a total

Rule 6

Current page should have 2 page links available either side of it.

Unless current page is:

  • page 1
  • page 2
  • the last page
  • the penultimate page
  • or if the total page count is less than 5
